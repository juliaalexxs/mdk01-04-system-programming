Выполнила: студентка Варламова Юлия Александровна, группы №24

О работе: Написала программу на C#, которая создаёт массив из 100 000 случайных чисел. Затем считает сумму этих чисел двумя способами. Через обычный цикл for, который работает последовательно, одно число за другим и через цикл Parallel.For, который работает параллельно, используя несколько потоков.
Программа выводит в консоль сумму и время выполнения каждого способа, чтобы можно было сравнить производительность.
